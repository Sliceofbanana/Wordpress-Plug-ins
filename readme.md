# WordPress Plugins Collection

A curated collection of custom WordPress plugins created by Brent and Genesis Jr.

## Overview

This repository serves as a centralized location for custom WordPress plugins designed to extend and enhance WordPress functionality. Each plugin is built with security, performance, and ease of use in mind.

## Installation

### General Installation Steps

1. Download the desired plugin file from this repository
2. Upload the plugin file to your WordPress site's `/wp-content/plugins/` directory
3. Navigate to the WordPress Admin Dashboard
4. Go to **Plugins** → **Installed Plugins**
5. Find the plugin in the list and click **Activate**

### Alternative Installation (via WordPress Admin)

1. Download the plugin file from this repository
2. In WordPress Admin, go to **Plugins** → **Add New**
3. Click **Upload Plugin** at the top of the page
4. Choose the downloaded file and click **Install Now**
5. After installation, click **Activate Plugin**

## Usage

Each plugin includes specific shortcodes, settings pages, or functionality that can be accessed through the WordPress Admin Dashboard. Refer to individual plugin documentation for detailed usage instructions.
